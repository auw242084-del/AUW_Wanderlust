# AUW Wanderlust 🌍
A travel planning web application built for AUW students.

### Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLite
- **Frontend:** HTML5, CSS3

### How to Run
1. Run `python init_db.py` to setup the database.
2. Run `python app.py` to start the server.
3. Open `http://127.0.0.1:5000` in your browser.
